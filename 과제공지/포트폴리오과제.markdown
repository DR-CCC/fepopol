# 금융공학 과제 정리

> 원문: `포트폴리오과제.pdf`  
> 주제: 포트폴리오 구성을 통한 효율적 자본시장선 구현

## 1. 과제 개요

이 과제는 주식 또는 ETF 수익률 데이터를 이용해 정규성 검정을 수행하고, 여러 자산으로 구성된 포트폴리오를 평균-분산 선택법 관점에서 최적화한 뒤, 효율적 투자선과 효율적 자본시장선을 구현하는 것을 목표로 한다.

주요 흐름은 다음과 같다.

1. 금융 이론에서 정규분포와 선형성 가정이 왜 중요한지 이해한다.
2. 자산 수익률의 정규성을 시각화와 통계적 검정으로 확인한다.
3. 여러 자산으로 포트폴리오를 구성한다.
4. 샤프 지수 최대화 포트폴리오와 최소 변동성 포트폴리오를 찾는다.
5. 효율적 투자선과 무위험 자산을 포함한 효율적 자본시장선을 그린다.

## 2. 배경: 금융 이론과 정규분포 가정

금융 이론에서 중요한 통계 요소 중 하나는 **정규분포**와 **선형성**이다. 아래 이론들은 모두 주식 시장의 수익률이 정규분포를 이룬다고 가정한다.

| 금융 이론 | 설명 |
|---|---|
| 포트폴리오 이론 | 주식 수익률이 정규분포를 이룰 경우, 최적 포트폴리오 선택 문제는 각 주식의 평균 수익률, 수익률 분산(변동성), 주식 사이의 공분산을 고려하는 문제로 정리할 수 있다. |
| 평균-분산 선택법 | 포트폴리오 선택 시 수익률의 평균(기대 수익)과 분산(위험)을 기준으로 최적 포트폴리오를 찾는 방법이다. 수학적 최적화 문제로 접근한다. |
| 효율적 투자선 | 동일한 위험 수준에서 가장 높은 수익률을 주거나, 동일한 수익률에서 가장 낮은 위험을 가진 포트폴리오 조합들의 경계선이다. |
| 자본 시장선 | 무위험 자산을 포함한 포트폴리오의 최적 조합을 나타내는 선이다. 샤프 지수 최대 포트폴리오와 무위험 자산을 잇는 직선이다. |

## 3. 통계 분석: 정규성 검정

정규성을 검정하기 위해서는 다음 두 절차를 진행한다.

### 3.1 1단계: 시각화

시각화 단계에서는 수익률 분포가 정규분포와 유사한지 눈으로 확인한다.

- 히스토그램과 정규분포의 밀도 함수를 비교한다.
- Q-Q Plot을 사용한다.
- Q-Q Plot은 데이터의 분위수와 정규분포상의 분위수를 비교한 그래프다.

PDF에 포함된 시각화 예시는 다음 두 가지다.

- 히스토그램과 정규분포의 밀도 함수 비교
- Q-Q Plot

### 3.2 2단계: 통계적 검정 절차

통계적 검정 단계에서는 검정 통계량과 P-value를 통해 정규성을 판단한다.

- 왜도 검정: 분포의 비대칭도 통계량을 이용해 데이터의 왜도가 정규분포와 일치하는지 판단한다.
- 첨도 검정: 분포의 뾰족함 또는 완만함을 나타내는 통계량을 이용해 데이터의 첨도가 정규분포와 일치하는지 판단한다.
- 정규성 검정: 왜도 검정과 첨도 검정을 조합해 정규성 여부를 검정한다.

## 4. 정규성 검정 예시: S&P 500 ETF 주가 데이터

PDF에서는 S&P 500 ETF 주가 데이터인 **SPY**를 기반으로 정규성 검정을 실시한다.

분석 대상은 다음과 같다.

- 로그 정규분포 값
- 로그 수익률

SPY 데이터는 다음 이유로 정규성을 만족하지 않는 것으로 확인된다.

1. Q-Q Plot에서 데이터가 직선을 따르지 않는다.
2. Fat Tail 현상이 나타난다.
3. 통계량 검증 시 P-value가 0에 가까워 귀무가설을 기각한다.
4. 여기서 귀무가설은 통계량이 정규분포를 따른다는 가정이다.

PDF에 포함된 예시 그림은 다음과 같다.

- SPY Q-Q Plot
- SPY 데이터의 통계량 검증

## 5. 포트폴리오 최적화 개요

포트폴리오의 목표는 다음 두 가지다.

1. 목표 수익률이 주어졌을 때 포트폴리오 위험을 최소화한다.
2. 일정 위험이 주어졌을 때 포트폴리오 수익률을 극대화한다.

대표적인 포트폴리오 최적화 방법론은 **평균-분산 선택법**이다.

평균-분산 선택법은 기대 수익률과 리스크(변동성)의 관계를 고려해 최적의 포트폴리오를 구성한다. PDF에서는 투자자가 매수 포지션만 가능하다고 가정한다. 즉, 모든 자산 비중은 매수에 사용되며 공매도는 고려하지 않는다.

### 5.1 포트폴리오 수익률 기대값

포트폴리오 기대 수익률은 각 자산의 비중과 각 자산의 기대 수익률을 곱한 값을 모두 더해 계산한다.

```math
\mu_p = \sum_i w_i E[r_i] = \sum_i w_i \mu_i
```

의미는 다음과 같다.

- 미래 기대 수익률에 대한 기대값이다.
- 각 자산 비중의 합은 1이 되어야 한다.

```math
\sum_i w_i = 1
```

### 5.2 포트폴리오 분산 기대값

포트폴리오 분산의 기대값은 공분산행렬을 이용해 계산한다.

```math
\sigma_p^2 = E[(r - \mu)^2] = \sum_{i \in I} \sum_{j \in I} w_i w_j \sigma_{ij}
```

기호의 의미는 다음과 같다.

| 기호 | 의미 |
|---|---|
| `w_i` | 각 자산의 비중 |
| `\mu_i` | 각 자산의 기대 수익률 |
| `\sigma_{ij}` | 자산 `i`와 자산 `j` 사이의 공분산 |
| `\mu_p` | 포트폴리오 기대 수익률 |
| `\sigma_p^2` | 포트폴리오 분산 |

## 6. 포트폴리오 구성 대상과 샤프 지수

PDF에서는 총 4개의 자산을 대상으로 포트폴리오 최적화를 진행한다.

- 애플
- 마이크로소프트
- S&P 500
- Gold ETF

샤프 지수는 투자 포트폴리오의 **위험 조정 수익률**로, 투자 성과를 평가하기 위한 주요 지표다.

```math
SR = \frac{\mu_p - r_f}{\sigma_p}
```

기호의 의미는 다음과 같다.

| 기호 | 의미 |
|---|---|
| `SR` | 샤프 지수 |
| `\mu_p` | 포트폴리오의 기대 수익률 |
| `r_f` | 무위험 수익률 |
| `\sigma_p` | 포트폴리오 수익률의 표준편차 |

무위험 수익률이 0일 때 샤프 지수는 다음과 같이 해석할 수 있다.

```math
SR = \frac{\mu_p}{\sigma_p}
```

## 7. 포트폴리오 최적화 결과 예시

앞선 시뮬레이션 결과를 기반으로 포트폴리오 최적화를 진행한다. PDF에서는 Optimize 라이브러리를 활용한 예시가 포함되어 있다.

최적화 대상은 다음 두 가지다.

1. 샤프 지수를 최대화하는 포트폴리오
2. 최소 변동성 포트폴리오, 즉 포트폴리오 분산이 최소화되는 포트폴리오

최적화 예시 코드에는 다음 구성 요소가 포함된다.

- 제약 조건: 등식 제한조건
- 최소화할 함수: 샤프 지수 최대화 문제를 최소화 문제로 바꾸기 위해 음수화
- 최적화 진행 및 결과 확인

PDF에 제시된 결과는 다음과 같다.

| 구분 | 샤프 지수 최대화 포트폴리오 | 최소 변동성 포트폴리오 |
|---|---:|---:|
| 비중 | `[0.319, 0.059, 0.5, 0.122]` | `[0.544, 0.442, 0, 0.014]` |
| 수익률 | 15.9% | 6.4% |
| 변동성 | 17.5% | 11.1% |
| 샤프 지수 | 91.0% | 57.9% |

## 8. 효율적 투자선

효율적 투자선은 여러 투자 포트폴리오 중 최적 포트폴리오의 집합이다.

효율적 투자선에 포함되는 포트폴리오는 다음 조건을 만족한다.

1. 목표 수익 수준 대비, 즉 같은 수익률 대비 최소 변동성을 가진다.
2. 목표 위험 수준 대비, 즉 같은 위험률 대비 최대 수익률을 가진다.

PDF의 그림은 수익률을 고정하고 최소 변동성을 가진 포트폴리오 비중에 대한 시뮬레이션 결과를 보여준다.

그림에 포함된 주요 표시 요소는 다음과 같다.

- 효율적 투자선
- 투자선 중 최대 샤프 지수 지점
- 투자선 중 최소 위험, 즉 최소 변동성 또는 최소 분산 지점

## 9. 효율적 자본시장선

무위험 자산이 존재할 경우에는 효율적 투자선에 무위험 자산을 포함한 포트폴리오를 고려한다. 이를 효율적 자본시장선으로 정리할 수 있다.

효율적 자본시장선은 다음 특징을 가진다.

- 무위험 자산의 위험-수익률 지점을 지난다.
- 효율적 투자선에 접하는 포트폴리오를 연결한다.
- 무위험 자산과 접점 포트폴리오를 잇는 직선으로 표현된다.

PDF에서는 효율적 자본시장선을 계산하기 위해 다음 방법이 필요하다고 설명한다.

1. 효율적 투자선의 함수 형태를 근사한다.
2. 효율적 투자선의 1차 도함수 근사치를 구한다.
3. 큐빅 스플라인 보간법을 활용한다.

PDF에 포함된 최적화 결과 과정은 다음과 같다.

1. 큐빅 스플라인 보간을 활용한 효율적 투자선 근사
2. 효율적 자본시장선 계산

PDF의 예시 그림은 **무위험 이자율 1%**인 경우의 효율적 자본시장선을 보여준다.

## 10. 과제 수행 체크리스트

PDF 내용을 기준으로 과제 수행 시 포함해야 할 핵심 항목은 다음과 같다.

- 자산별 가격 데이터를 수집한다.
- 자산별 로그 정규분포 값과 로그 수익률을 계산한다.
- 각 자산 수익률에 대해 정규성 검정을 수행한다.
- 히스토그램과 정규분포 밀도 함수를 비교한다.
- Q-Q Plot을 작성한다.
- 왜도 검정, 첨도 검정, 정규성 검정을 수행한다.
- 여러 자산으로 포트폴리오를 구성한다.
- 각 자산의 기대 수익률, 변동성, 공분산행렬을 계산한다.
- 매수 포지션만 가능하다는 제약을 둔다.
- 포트폴리오 비중의 합이 1이 되도록 제약한다.
- 포트폴리오 기대 수익률을 계산한다.
- 포트폴리오 분산과 표준편차를 계산한다.
- 샤프 지수를 계산한다.
- 샤프 지수 최대화 포트폴리오를 찾는다.
- 최소 변동성 포트폴리오를 찾는다.
- 시뮬레이션 결과를 기반으로 효율적 투자선을 작성한다.
- 효율적 투자선에서 최대 샤프 지수 지점과 최소 위험 지점을 표시한다.
- 무위험 수익률 1%를 가정해 효율적 자본시장선을 계산하고 Plotting한다.
- 효율적 투자선을 근사할 때 큐빅 스플라인 보간법을 활용한다.
- 최종적으로 가장 효율적인 포트폴리오를 구성하고, 그 구성 내역과 구성 논리를 설명한다.
- 모든 핵심 항목들에 대하여 PPT slide 발표 자료를 만들 수 있을 정도로 자세하게 진행 내역들을 정리하여 제시한다.
